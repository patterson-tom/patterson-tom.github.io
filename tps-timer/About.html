Scramble Generator for NxNxN puzzles

This tool generates random move scrambles for any NxNxN puzzle (e.g. 3x3x3 or 5x5x5). You can alter the size of the puzzle by changing the 'Puzzle size' field. In this textbox specify the number of layers of the puzzle. For example, to generate a 3x3x3 scramble, type '3'. You can change the number of moves in the scramble by altering the 'Scramble length' field.

The random state scramble conforms to the following rules:

  • Consecutive moves cannot be on the same face. This prevents redundant sequences such as R R'
  • Consecutive moves on opposite faces can only occur in one order (R L / U D / F B). This also prevents redundant sequences such as R L R' L'
  • For even layered cubes, the highest layer wide turns can only occur on the R, F and U faces. This is because their L, D and B counterparts are not needed to be able to effectively make every possible move.

The notation used for the scrambles is standard Speedcubing Notation as denoted in the official World Cube Association (WCA) regulations.

A visual representation of the scramble is provided, with the standard colour scheme (Up: white, Down: yellow, Left: orange, Right: red, Front: green, Back: blue). The scramble is performed with green on the front face and white on top, as per the WCA regulations.

There is also a rudimentary timer on the tool. To start the timer, press the space key. When you release this key, the timer will start. To stop the timer again, press the space bar once more. The timer will automatically reset when you go to start it once more.
